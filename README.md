# Unity Request Queue Demo

Небольшое Unity-приложение, демонстрирующее работу с бэкендом через **очередь последовательных запросов** с поддержкой отмены, а также чистую архитектуру на Zenject.

## Стек

- Unity 2022.3+
- Zenject (DI)
- UnityWebRequest
- UniTask, UniRx, DOTween
- ScriptableObject для конфигурации
- Паттерн MVC/MVP, Object Pool

## Возможности

Приложение из трёх вкладок с нижней навигацией:

1. **Кликер** — начисление валюты по тапу и автосбором, система энергии, VFX (партиклы, анимация, звук), параметры из ScriptableObject.
2. **Погода** — периодические запросы к Weather API с отменой при уходе с вкладки.
3. **Породы собак** — список пород и факты по выбранной породе через Dog API, попап с адаптивной высотой.

## Ядро: очередь запросов

Сетевые запросы выполняются строго последовательно через единую очередь:

- добавление запроса в очередь и получение результата;
- отмена выполняющегося запроса;
- удаление запроса из очереди, если он ещё не стартовал.

Реализация на UniTask + CancellationToken.
